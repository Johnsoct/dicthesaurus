## Command Flags

`help: -h, -help, --help`


## Subcommand Flags

`-d` : Display only dictionary results
`-e` : Display the word in a sentence
`-ss` : Display format is "sweet and simple"
`t` : Display only thesaurus results
`ud` : Also query Urban Dictionary for results
